Aprende a usar el sistema

Para comenzar a utilizar las funcionalidades que brinda el sistema es necesario crear un usuario, para esto nos dirigimos al apartado de Iniciar Sesión y creamos un usuario.

Una vez que tengamos nuestro usuario creado iniciaremos sesión en la misma pestaña de Iniciar Sesión y damos clic en Entrar y nos abrirá una interfaz exclusiva para hacer uso del sistema.

En la parte de la derecha aparece un menú que contiene los botones de: "Inicio", "Gráficas", "Perfil", "Archivos" y por último "Documentación".


En el apartado de Inicio nos mmuestra una breve información sobre qué trata nuestro sistema, para qué sirve y cómo funciona.

Como siguiente apartado tenemos Gráficas la cual es una de las más interesantes, ya que aquí podemos seleccionar nuestro archivo con el cual podemos buscar patrones y hacer predicciones del mismo.


También podemos interactuar con nuestra gráfica seleccionando el tipo de gráfica que deseamos, así como también seleccionar una cierta parte de la gráfica para ver más a detalle el patrón, hacer zoom, alejar y descargar como archivo png nuestrá gráfica

En la parte de Perfil nos muestra nuestra información personal como nombre, apellidos, teléfono, correo electrónico y número de archivos que hemos subido.

En la parte de Archivos podemos subir un archivo con el cual queremos trabajar para posteriormente envíar datos. También podemos editar ó eliminar los archivos que hemos subido.

Por último tenemos el apartado de Documentación el cual muestra la documentación de nuestro proyecto como los repositorios Github de nuestro código fuente, nuestro protocolo de Trabajo Terminal, el manual técnico y el manual de usuario.

Por último para regresar a la parte de inicio del sitio damos clic en la parte superiro derecha donde dice Usuario y damos clic en Cerrar Sesión y listo, esto es todo lo escencial que debemos saber para hacer un buen uso del sitio.
